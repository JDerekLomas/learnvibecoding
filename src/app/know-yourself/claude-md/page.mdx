# Your CLAUDE.md

Your CLAUDE.md file isn't a configuration file. It's a portrait of how you think and work.

Start with this template and make it yours:

```markdown
# Who I Am

## How I Work
- [Your general approach: fast and iterative? Careful and methodical?]
- [What matters most: shipping fast? Getting details right? Beautiful design?]

## My Aesthetic
- [What "good" looks like to you]
- [Sites/apps you admire]
- [Words that describe your style]

## Preferences
- Be concise / Be detailed (pick one)
- Explain before doing / Just do it (pick one)
- Suggest alternatives / Follow instructions exactly (pick one)

## Technical
- Deploy to Vercel with `vercel --prod`
- [Your preferred tech stack, if any]
- [Your GitHub username]
```

This isn't set-and-forget. Update it as you learn more about how you work. Every few weeks, revisit it. Your CLAUDE.md grows as you grow.
